# VSB 파일 변환

### 들어가며...

.vsb(Vehicle Spy Binary)파일은 Vehicle Spy 3에서 기본적으로 사용하는 로그 파일 형식입니다. 
이 파일 형식은 다른 로그 파일과 비교하여 용량이 작다는 장점이 있습니다. 
VSB 파일은 필요한 경우 .csv, .mdf, 등 다양한 파일 포맷으로 변환하여 사용할 수 있습니다.

이번 장에서는 VSB 파일을 .csv, .blf 등 다른 메시지 포맷으로 변환하거나, .dat, .mf4 등 시그널 포맷으로 변환하는 방법에 대해 알아보겠습니다.
